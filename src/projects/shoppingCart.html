Shopping Cart

Made in React, this is a Website with a Home and a Shop page, with the Cart and Payout Section, where you can buy 'fake' Products fetched from Fake Store API.

This is a Single-Page Application, made possible with React Router which is used in routes.jsx and in the Components where's <Link>, so you can navigate all the pages without loading again the Website

State helps with 'remembering' which Product was added to the Cart and their quantity.
